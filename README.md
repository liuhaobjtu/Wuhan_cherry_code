# 文件运行说明

```
new2.mp4是武汉大学代码生成的视频(就是new.mp4视频)

video.avi -> pic文件夹中的图片 -> new文件夹中的图片 -> new.mp4视频

cv_2.py：从video.avi视频中提，并保存到取出每一帧图片，并保存到pic文件夹中；
cv_3.py：把pic文件夹中的图片，把像素按10*10间隔替换成武汉加油，并保存到new文件夹中；
cv_4.py：把new文件夹中的图片，按一定视频格式，帧率，宽高尺寸生成new.mp4视频；

```
